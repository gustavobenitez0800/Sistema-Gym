IMPORTANTE: LEER ANTES DE SUBIR A GITHUB
========================================

Ya he configurado el sistema para que ignore tus claves privadas, PERO si ya habías subido el archivo 'src/config.js' anteriormente, necesitas ejecutar un comando especial para "olvidarlo" sin borrarlo de tu computadora.

PASOS PARA TAPAR EL CONTENIDO SENSIBLE:

1. Abre tu terminal (donde usas git).
2. Ejecuta este comando EXACTO:
   
   git rm --cached src/config.js

   (Esto le dice a git: "deja de rastrear este archivo", pero lo MANTIENE en tu carpeta local para que el sistema siga funcionando).

3. Luego haz tus comandos normales:
   git add .
   git commit -m "Mejoras de seguridad"
   git push

----------------------------------------
¿POR QUÉ ESTO ES SEGURO?
1. Tu archivo 'src/config.js' (con las claves) se queda en TU PC.
2. El archivo 'src/config.example.js' (sin claves) es para que otros sepan qué crear.
3. El archivo '.gitignore' asegura que git no vuelva a intentar subir tus claves.
4. El archivo 'ACTUALIZAR.bat' y 'update-exclude.txt' aseguran que cuando actualices en el futuro, no se sobrescriba tu config local.

El sistema está listo y protegido.
